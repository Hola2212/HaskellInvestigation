## ¿Qué realiza?

1) `main :: IO ()`  
   Indica que será una entrada/salida.

2) `main = putStrLn "Hello, World!"`  
   Define la función `main`, que ejecuta una acción de entrada/salida: `putStrLn`,  
   que toma una cadena, la imprime en la consola y añade un salto de línea.
